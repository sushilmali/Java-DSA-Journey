# Day 4 - Time & Space Complexity

## Time Complexity
Time complexity tells us how the running time of an algorithm grows as the input size increases.

## Big O
Represents the worst-case time complexity.

## Big Omega (Ω)
Represents the best-case time complexity.

## Big Theta (Θ)
Represents the average-case time complexity.

## Common Time Complexities

O(1)
O(log n)
O(√n)
O(n)
O(n log n)
O(n²)
O(n³)
O(n!)
O(nⁿ)

Better Performance

O(1) > O(log n) > O(n) > O(n log n) > O(n²)
